    Interface ICoordinateFilter

    An interface for classes which use the values of the coordinates in a Geometry. Coordinate filters can be used to implement centroid and envelope computation, and many other functions.

    ICoordinateFilter is an example of the Gang-of-Four Visitor pattern.

    Note: it is not recommended to use these filters to mutate the coordinates. There is no guarantee that the coordinate is the actual object stored in the geometry. In particular, modified values may not be preserved if the target Geometry uses a non-default CoordinateSequence. If in-place mutation is required, use ICoordinateSequenceFilter.
