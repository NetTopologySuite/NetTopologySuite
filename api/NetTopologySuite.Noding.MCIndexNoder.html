    Class MCIndexNoder

    Nodes a set of ISegmentStrings using a index based on MonotoneChains and a ISpatialIndex<T>. The ISpatialIndex<T> used should be something that supports envelope (range) queries efficiently (such as a Quadtree" or STRtree<TItem>.

    The noder supports using an overlap tolerance distance. This allows determining segment intersection using a buffer for uses involving snapping with a distance tolerance.
